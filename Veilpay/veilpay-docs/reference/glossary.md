# Glossary

## Chain key

A stable network identifier used by Veilpay, such as `ethereum`, `base`, `solana`, or `stellar`.

## Encrypted note

Payment metadata encrypted for a recipient so memo content is not disclosed in plaintext.

## Nullifier

A privacy-pool value used to prevent double-spending without revealing the original note.

## RPC proxy

A backend route that forwards supported chain calls while keeping provider credentials server-side.

## Self-custody

A wallet model where private keys and mnemonics stay on the user's device and are never sent to the backend.

## SPP

Stellar Private Payments, the Stellar testnet privacy-pool integration track.

## Stealth address

A one-time address derived to reduce direct linkability between payer, recipient, and public identity.
