# Solana

Veilpay supports Solana wallet flows through the SVM address family.

## Mainnet and devnet

- Solana mainnet is used for production network flows.
- Solana Devnet is used for development and testing.

## Balance and send flows

Solana balance fetching uses Solana JSON-RPC semantics. Send flows use chain-specific signing and transaction submission utilities.

## WalletConnect

WalletConnect namespace support includes Solana session handling where supported by the app and dapp.

## Privacy roadmap

Solana is part of the supported chain set, but the native privacy-chain roadmap is currently centered on Stellar SPP first, followed by planned Monero, Zcash, and Midnight tracks.
